How to add a player?

To add a player, the first thing you have to do is to understand what kind of data you have to insert.

This platform, is based on the EA Sport Fifa21 ranking database, so in order to get the right data, connect to the Fifa21 DB by clicking here.

Once you've been connected, search for the player you want to add, for example, Messi.

All the data you need to fill the form and insert the player are in this screen. The only data you don't see immediately is the playerId. In this case you have two way to retrieve it:

  1. Read from the address bar of the browser the last string, that is a numeric code:
  2. Right click on the player image, open it in another tab of the browser and read from the address bar the last string that is a numeric code with an extension (e.g. 158023.png). Of course you have to exclude the extension and take only the numeric code.
